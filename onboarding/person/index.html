Person

Join the following

  • AAD groups - submit pull request to the DevOps Azure AD repository
- name: username
  groups:
  - DTS CFT Developers
  - <your-team-group> # check other team members to find the group

Where username is the part of your email before @ (usually firstname.lastname like john.smith), justice.gov.uk users need to include their full email, e.g. mary.jayne@justice.gov.uk.

These groups will give you access to the VPN, the Azure Portal and logs.

  • GitHub Access - Read the acceptable use policy, if you agree to follow it then ask someone in your team who has access to add you in the GitHub management repository by submitting a pull request.
  • GitHub Teams - ask your Technical Lead or a senior member of the team to add you to your GitHub team group.
  • Office 365 team
  • Add Github Slack user mapping
